## ⚙️ Setup & How to Run

### Prerequisites

- **Java** 11 or higher — [Download JDK](https://adoptium.net/)
- **Git** — [Download Git](https://git-scm.com/)

### Clone the Repository

```bash
git clone https://github.com/devraman07/DataStructures_And_Algorithms.git
cd DataStructures_And_Algorithms
```

### Compile & Run a File

```bash
# Compile
javac /datastructures/linkedlist/LinkedList.java

# Run
java -cp src datastructures.linkedlist.LinkedList
```

### Using an IDE (Recommended)

1. Open the project in **IntelliJ IDEA** or **Eclipse**
2. Mark `src/` as the Sources Root
3. Run any file directly using the IDE's run button

## 📊 Time & Space Complexity Cheat Sheet

| Data Structure | Access | Search | Insert | Delete |
|---|---|---|---|---|
| Array | O(1) | O(n) | O(n) | O(n) |
| Linked List | O(n) | O(n) | O(1) | O(1) |

## 🤝 Contributing

Contributions are welcome! Here's how to get started:

1. **Fork** the repository
2. **Create a branch** for your feature or fix
   ```bash
   git checkout -b feature/add-trie-implementation
   ```
3. **Write clean, commented Java code** — each file should include:
   - A brief description of the data structure or algorithm at the top
   - Time and space complexity noted in comments
   - At least one example in a `main()` method demonstrating usage
4. **Commit** your changes with a clear message
   ```bash
   git commit -m "Add iterative DFS for graph traversal"
   ```
5. **Open a Pull Request** — describe what you've added and why

### Contribution Guidelines

- Follow standard Java naming conventions (`camelCase` for methods, `PascalCase` for classes)
- Keep one topic per file where possible
- Do not copy solutions directly from LeetCode or other platforms — original implementations only
- All PRs are reviewed before merging
